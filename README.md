## Installing
`jasmine-es6` is available as an
[npm package](https://www.npmjs.com/package/jasmine-es6).

## Development
### Getting Started
The application requires the following external dependencies:
* Node.js

The rest of the dependencies are handled through:
```bash
npm install
```

Run tests with:
```bash
npm test
```
